Proxy Checker API v1.3

This is a proxy checker API running on Cloudflare Pages.

Usage

Send a GET request to /check with the following parameters:

  • ip: IP address and port (e.g., 1.2.3.4:8080). Multiple IPs can be separated by commas (max 10).
  • format: (Optional) json (default), text, or csv.

Example

GET /check?ip=129.150.58.86:57621,2.59.183.233:444&format=json
